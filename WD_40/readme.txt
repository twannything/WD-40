2.1 Create BigInt, Delete BigInt, Zerorize BigInt

생성되어진 BigInt의 부호, 워드 길이, 주소 값을 확인한다.

2.2 Set BigInt by array

배열로 BigInt를 저장하는 경우, 배열의 마지막 값은 Bigint 멤버의 a의 첫 번째 배열로 할당한다.

array로 BigInt를 구현 후 생성되어진 BigInt의 부호, 워드 길이, 주소 값을 확인한다.
